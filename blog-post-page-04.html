ກັບຄືນໜ້າລາຍການ

ວິທີການຂຽນສູດຄິດໄລ່ໃນຕາຕະລາງຊໍາລະເງິນກູ້ຮູບແບບຕ່າງໆ

04. ເລື່ອນຊໍາລະຕົ້ນທຶນ 3 ເດືອນທໍາອິດ. (ຍັງບໍ່ທັນໄດ້ແປງເນື້ອໃນ)

→ ເດືອນ / ປີ = IF(ໄລຍະເວລາກູ້ຢືມ < ຄັ້ງທີຊໍາລະ," ",EDATE(ວັນທີກູ້ຢືມ,ຄັ້ງທີຊໍາລະ))
    ໝາຍເຫດ:  ຄັ້ງທີຊໍາລະ ແມ່ນເລີ່ມຕັ້ງແຕ່ຄັ້ງທີ 1 2 3....ໄປຈົນຮອດຄັ້ງທີຊໍາລະເດືອນສຸດທ້າຍ.

→ ຕົ້ນທຶນຕໍ່ເດືອນ:

−  ຕົ້ນທຶນຕໍ່ເດືອນ 1 = IF(ວົງເງິນກູ້ <=0," ",IF(ວົງເງິນກູ້ >=0,CEILING(ວົງເງິນກູ້ / ໄລຍະເວລາກູ້ຢືມ,500),IF(ໄລຍະເວລາກູ້ຢືມ <=0," ")))
−  ຕົ້ນທຶນຕໍ່ເດືອນ 2 = CEILING(IF(ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 1 <= 0,0,ວົງເງິນກູ້ / ໄລຍະເວລາກູ້ຢືມ),500)
−  ຕົ້ນທຶນຕໍ່ເດືອນ 3 = CEILING(IF(ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 2 <= 0,0,IF(ຄັ້ງທີຊໍາລະ = ໄລຍະເວລາກູ້ຢືມ,ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 3,ວົງເງິນກູ້ / ໄລຍະເວລາກູ້ຢືມ),500)

−  ຕົ້ນທຶນຕໍ່ເດືອນ 4 = CEILING(IF(ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 3 <= 0,0,IF(ຄັ້ງທີຊໍາລະ = ໄລຍະເວລາກູ້ຢືມ,ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 4,ວົງເງິນກູ້ / ໄລຍະເວລາກູ້ຢືມ),500)
−  ຕົ້ນທຶນຕໍ່ເດືອນ 5 = CEILING(IF(ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 4 <= 0,0,ວົງເງິນກູ້ / ໄລຍະເວລາກູ້ຢືມ),500)
−  ຕົ້ນທຶນຕໍ່ເດືອນ 6 = CEILING(IF(ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 5 <= 0,0,IF(ຄັ້ງທີຊໍາລະ = ໄລຍະເວລາກູ້ຢືມ,ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 5,ວົງເງິນກູ້ / ໄລຍະເວລາກູ້ຢືມ),500)

→ ດອກເບ້ຍຕໍ່ເດືອນ:

−  ດອກເບ້ຍຕໍ່ເດືອນ 1 = CEILING(ວົງເງິນກູ້ x ອັດຕາດອກເບ້ຍ,500)
−  ດອກເບ້ຍຕໍ່ເດືອນ 2 = IF(ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 1 <= 0,0,ດອກເບ້ຍຕໍ່ເດືອນ ເດືອນທີ 1)
−  ດອກເບ້ຍຕໍ່ເດືອນ 3 = IF(ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 2 <= 0,0,ດອກເບ້ຍຕໍ່ເດືອນ ເດືອນທີ 1 ຫຼື 2 ກໍ່ໄດ້)

−  ດອກເບ້ຍຕໍ່ເດືອນ 4 = CEILING(ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 3 x ອັດຕາດອກເບ້ຍ,500)
−  ດອກເບ້ຍຕໍ່ເດືອນ 5 = IF(ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 4 <= 0,0,ດອກເບ້ຍຕໍ່ເດືອນ ເດືອນທີ 4)
−  ດອກເບ້ຍຕໍ່ເດືອນ 6 = IF(ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 5 <= 0,0,ດອກເບ້ຍຕໍ່ເດືອນ ເດືອນທີ 4 ຫຼື 5 ກໍ່ໄດ້)

→ ລວມຊໍາລະຕໍ່ເດືອນ = SUM(ຕົ້ນທຶນຕໍ່ເດືອນ : ດອກເບ້ຍຕໍ່ເດືອນ)

→ ຍອດເຫຼືອເງິນກູ້:

−  ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 1 = ວົງເງິນກູ້ − ຕົ້ນທຶນຕໍ່ເດືອນທີ 1
−  ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 2 = IF(ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 1 − ຕົ້ນທຶນຕໍ່ເດືອນທີ 2 <= 0,0,ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 1 − ຕົ້ນທຶນຕໍ່ເດືອນທີ 2)
−  ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 3 = IF(ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 2 − ຕົ້ນທຶນຕໍ່ເດືອນທີ 3 <= 0,0,ຍອດເຫຼືອເງິນກູ້ ເດືອນທີ 2 − ຕົ້ນທຶນຕໍ່ເດືອນທີ 3)
